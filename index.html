Sobre

Em 2007, obtive minha graduação em Direito e logo passei na OAB. Trabalhei como advogada e como professora de inglês até que, em 2013, a minha vida profissional mudou completamente. Por hobby, criei o blog de viagens Projeto 101 Países, onde narro minhas aventuras pelas dezenas de países por onde passei. Além proporcionar viagens maravilhosas, o blog também me apresentou a um novo mundo: o da Programação. Na curiosidade de mudar layouts do blog, dei meus primeiros passos em HTML e CSS até que, após algum tempo, resolvi fazer um college de Computer Programming no Canadá. Passei pouco mais de um ano morando fora do Brasil, aprimorando meu inglês, apredendo várias linguagens de programação e novas tecnologias. Atualmente, resido no Brasil e, de vez em quando, passeio por algum lugar do mundo ;)

Paper, Scissors, Lizard, Spock Game

Web Programming

Trabalho acadêmico com o objetivo de desenvolver o clássico jogo "Pedra, Papel e Tesoura", incluindo mais dois gestos: Lagarto e Spock, este último inspirado no personagem de Star Trek. Todo o desenvolvimento foi feito no editor de código Visual Studio Code (VS Code), criando uma aplicação web interativa e responsiva.

Bus Ticket Booking App

Mobile Web Based Application

Desenvolver um aplicativo para venda de passagens de ônibus que inclua: quatro cidades de partida e chegada, seleção de empresas de ônibus com radio buttons, calendário para datas, seleção de até 10 passagens, lista de horários dos ônibus com destaque para horários selecionados, campos para nome, email, gênero e idade (mínimo 18 anos). Inclui botão para enviar a solicitação, mostrando os detalhes da reserva em uma nova página. Utiliza localStorage e JavaScript conforme ensinado. Mobile-friendly com header e footer mostrando o nome do aluno usando template literals.

Bus Ticket Booking App

Mobile Web Based Application

Develop an application for bus ticket sales that includes: four cities for departure and arrival, selection of bus companies using radio buttons, a calendar for dates, the ability to select up to 10 tickets, a list of bus schedules with highlighted selected times, and fields for name, email, gender, and age (minimum 18 years). The application also features a button to submit the request, which will display the reservation details on a new page. It makes use of localStorage and JavaScript as taught. The design is mobile-friendly, with a header and footer that show the student's name using template literals.
